Features
›
Repos
GitHub: unlimited public, limited private on free. BlackRoad: unlimited everything. 239 repos on Gitea. No limits.
›
CI/CD
GitHub Actions: 2,000 min/mo free, then $0.008/min. BlackRoad: unlimited. act_runner on fleet nodes. Zero per-minute cost.
›
Privacy
GitHub: Microsoft owns your code history, stars, issues, PRs. Trains Copilot on public repos. BlackRoad: your code stays on your hardware.
›
Issue Tracking
Both have issues, labels, milestones, projects. GitHub has Discussions. Gitea has equivalent functionality.
›
Code Review
Both have PR review with inline comments. GitHub has more automation (CODEOWNERS, required reviews). Gitea covers basics.
›
Packages
GitHub: Container registry, npm, Maven. BlackRoad: Docker registry on Octavia. MinIO for artifacts.
›
Pages
GitHub Pages: free static hosting. BlackRoad: Caddy serves 151 domains. More flexible routing.
›
Uptime
GitHub: 99.9% SLA (paid). BlackRoad: best-effort on Pi hardware. But YOU control the downtime.
›
Verdict
GitHub wins on features and ecosystem. BlackRoad wins on sovereignty, cost, and privacy. Use both — Gitea primary, GitHub mirror.
